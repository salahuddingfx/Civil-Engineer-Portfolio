# Engr. Alam Ashik Portfolio

A premium, localized (EN/BN), MERN-stack portfolio for **Engr. Alam Ashik**, a professional Civil Engineer based in Cox's Bazar, Bangladesh.

## 🚀 Features

- **Dynamic MERN Stack**: Built with MongoDB, Express, React, and Node.js.
- **3D Architectural Visualization**: Featuring a procedurally generated, interactive 3D building model (React Three Fiber).
- **Dual Localization**: Full support for English and Bengali (EN/BN) with context-aware translations.
- **Premium UI/UX**: Sophisticated dark/light mode, GSAP-powered scroll animations, and a high-end design system.
- **Admin Dashboard**: Full content management for projects, gallery, and contact details.
- **SEO Optimized**: Advanced SEO strategies including 120+ keywords, localized meta tags, and high-performance assets.
- **High Performance**: Optimized Core Web Vitals including deferred 3D loading and preloaded assets.

## 🛠️ Tech Stack

- **Frontend**: Vite, React, TailwindCSS, GSAP, Three.js, React Three Fiber.
- **Backend**: Node.js, Express, JWT Authentication, Cloudinary Image Storage.
- **Database**: MongoDB (Mongoose).
- **SEO**: Dynamic Meta tags, automated Sitemap.xml and Robots.txt.

## 📁 Structure

- `/frontend`: React application (Vite).
- `/backend`: Node.js/Express API.

## 🔧 Installation

1. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. **Backend**:
   ```bash
   cd backend
   npm install
   # Configure your .env file
   npm run dev
   ```
